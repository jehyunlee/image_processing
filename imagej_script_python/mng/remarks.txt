  python 3 이후 문법인 print()로 작동. 2.7대의 print 문으로는 작동하지 않음.
  python 3 이후에 도입된 str.format 지원: '{}'.format()
  python 3.6 이후에 도입된 f-string은 미지원: f'{}'
